# Getting Started with Mycelix

Mycelix is a **living system for distributed intelligence** inspired by mycelial networks. Unlike traditional frameworks that treat computation as static processes, Mycelix embraces organic rhythms through its unique 28-day cycle.

## What is Mycelix?

Mycelix provides:

- **21 Living Primitives** - Reusable building blocks that understand time and context
- **4 Distinct Phases** - Dawn, Surge, Settle, and Rest shape system behavior
- **Distributed Coordination** - Nodes connect and collaborate like mycelial threads
- **Adaptive Intelligence** - The system learns and evolves with each cycle

## Core Philosophy

Traditional systems fight against time. Mycelix works *with* time:

```
Day 1-7:   Dawn   → Initialization, gentle awakening
Day 8-14:  Surge  → Peak activity, maximum throughput
Day 15-21: Settle → Consolidation, pattern recognition
Day 22-28: Rest   → Reflection, pruning, preparation
```
